# InsightFlow AI

AI-powered enterprise analytics platform for non-technical teams.

## Problem
Business teams experience delays accessing organizational data due to:
- fragmented permissions
- analyst dependency
- SQL knowledge barriers
- cross-timezone communication

## Solution
InsightFlow AI enables users to:
- query enterprise data using natural language
- discover datasets
- request permissions
- generate dashboards
- access KPI insights without SQL

## Features
- AI Query Assistant
- Dataset Discovery
- Dashboard Builder
- KPI Registry
- Governance Workflows
- Executive Insights

## Tech Stack
- Lovable
- Supabase
- OpenAI APIs
- AWS-inspired architecture

## Goal
Reduce enterprise data friction and democratize analytics access.
